# 🎫 osTicket: Ticket Lifecycle

## 📌 Project Overview

In this lab, I simulated the complete lifecycle of support tickets within the osTicket Help Desk system. This included creating tickets, assigning priorities, updating SLA plans, routing tickets to the appropriate departments, resolving issues, and closing tickets to simulate a real-world IT support workflow.

---

## 🎯 Objectives

- Create Support Tickets
- Assign Ticket Priorities
- Configure SLA Plans
- Assign Departments and Agents
- Update Ticket Status
- Resolve Tickets
- Close Tickets

---

## 🖥️ Technologies Used

- Microsoft Azure
- Windows 10
- Remote Desktop
- osTicket
- IIS (Internet Information Services)
